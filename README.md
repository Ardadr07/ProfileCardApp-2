## Proje Açıklaması
Bu proje React Native (Expo) ile geliştirilmiş “Gösterişli Profil Kartı” uygulamasıdır.  
Özel fontlar (Montserrat), tema sistemi (light/dark), gölgeli kart tasarımı, etkileşimli butonlar ve duyarlı (responsive) düzen kullanılmıştır.  
Ayrıca local bir Node.js server üzerinden profil verileri çekilerek listelenmektedir.

## Gereksinimler
- Node.js (LTS önerilir)
- npm
- Expo Go (fiziksel cihaz için)
